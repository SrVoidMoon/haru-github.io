Política de Privacidade

Última atualização: Junho de 2026

Esta Política de Privacidade descreve como o Haru coleta, utiliza, armazena e protege informações relacionadas ao uso do bot.

Ao utilizar o Haru, você concorda com as práticas descritas neste documento.

1. Informações Coletadas

O Haru pode coletar e armazenar determinadas informações necessárias para seu funcionamento, incluindo:

Dados do Discord

  • ID de usuários;
  • ID de servidores;
  • ID de canais;
  • ID de mensagens quando necessário para funcionalidades específicas.

Dados de Configuração

  • Canais autorizados para interação;
  • Configurações de perguntas diárias;
  • Regras personalizadas impostas pelos administradores;
  • Usuários marcados como especiais.

Dados de Conversa

Para fornecer contexto e melhorar a qualidade das respostas, o Haru pode armazenar partes das conversas realizadas com usuários.

Essas informações são utilizadas exclusivamente para:

  • Manter memória contextual;
  • Melhorar a continuidade das conversas;
  • Executar funcionalidades relacionadas à personalidade e interação do bot.

2. Como os Dados São Utilizados

Os dados coletados são utilizados para:

  • Operação normal do bot;
  • Responder mensagens e menções;
  • Gerenciar configurações do servidor;
  • Gerar perguntas diárias;
  • Manter contexto de conversa;
  • Garantir segurança e estabilidade do serviço.

O Haru não utiliza dados para publicidade direcionada.

3. Compartilhamento de Dados

O Haru não vende, aluga ou comercializa informações de usuários.

Entretanto, determinadas mensagens podem ser processadas por provedores externos de inteligência artificial utilizados pelo serviço para gerar respostas.

Esses provedores recebem apenas as informações necessárias para processar a solicitação enviada.

4. Armazenamento de Dados

As informações podem ser armazenadas localmente ou em serviços utilizados pelo desenvolvedor para operação do bot.

Os dados permanecem armazenados apenas pelo tempo necessário para o funcionamento dos recursos do Haru.

5. Exclusão de Dados

Administradores podem utilizar os comandos disponibilizados pelo Haru para remover determinadas informações armazenadas, incluindo memórias de conversa.

O desenvolvedor também poderá remover dados quando necessário para manutenção, segurança ou solicitação legítima.

6. Segurança

São adotadas medidas razoáveis para proteger os dados armazenados.

Apesar disso, nenhum sistema pode garantir segurança absoluta, e o uso do serviço ocorre por conta e risco do usuário.

7. Menores de Idade

O Haru não é destinado à coleta intencional de informações pessoais sensíveis de menores de idade.

Caso seja identificado armazenamento indevido de informações sensíveis, medidas poderão ser tomadas para sua remoção.

8. Alterações nesta Política

Esta Política de Privacidade poderá ser atualizada periodicamente para refletir mudanças no funcionamento do bot, requisitos legais ou melhorias do serviço.

A data da última atualização será sempre indicada no início deste documento.

9. Direitos dos Usuários

Os usuários podem solicitar esclarecimentos sobre os dados utilizados pelo Haru e, quando aplicável, solicitar sua remoção através dos canais oficiais de suporte do projeto.

10. Contato

Em caso de dúvidas sobre esta Política de Privacidade ou sobre o tratamento de dados realizado pelo Haru, entre em contato através dos canais oficiais disponibilizados pela equipe do projeto.

Ao utilizar o Haru, você declara ter lido e compreendido esta Política de Privacidade.
